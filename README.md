# Sports Data Science & Media Analytics Portfolio
By Kaylyn Schiemann

## Project 1: NHL Market Migration Analysis
**Goal:** Analyzed the move of an NHL franchise from Cable (RSN) to Scripps Broadcast (OTA).
- **Key Insight:** Standardized for market size, the move resulted in a **33.4% lift** in market penetration.
- **Tools:** Python, Google Trends API, Sentiment Analysis.

## Project 2: NASCAR Sponsorship Image Classification
**Goal:** Automated the detection of the "Blues Hog" logo in race footage using computer vision.
- **Key Insight:** Built a pipeline to distinguish between on-track racing and technical pit-stop imagery.
- **Tools:** TensorFlow, Python, OpenCV.

## Project 3: Predictive Clinical Intervention & ROI Analysis Engine
**Goal:** Developed a predictive model to identify heart failure patients at high risk of readmission and calculated the ROI of clinical interventions.
- **Key Insight:** Built a "Financial Impact" layer that translates model accuracy into cost-savings for healthcare providers.
- **Tools:** Python (Scikit-Learn), Logistic Regression/Random Forest, Cost-Benefit Analysis Modeling.
